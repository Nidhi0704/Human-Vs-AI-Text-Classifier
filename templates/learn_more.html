AI vs Human Text Classification Model

This machine learning model allows users to input a text, and it predicts whether the text was generated by AI or written by a human.

Real-World Impact

This model has a significant role in many areas, especially in online education systems. It helps in verifying whether students are genuinely writing their answers in subjective exams or if they are using AI-generated responses. By detecting the source of the text, we can ensure the authenticity of student submissions.

Machine Learning Techniques Used

We applied various machine learning algorithms to enhance the model's performance, including:

  • Naive Bayes Classifier
  • Decision Tree Classifier
  • Support Vector Classifier
  • Random Forest Classifier

Additionally, we used GridSearchCV to fine-tune the model and improve its accuracy, ensuring the best performance in identifying AI-generated text versus human-written content.
